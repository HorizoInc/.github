# タイムゲートラマン SNR 最適化レポートツール
<img width="900" height="794" alt="SNR optimize" src="https://github.com/user-attachments/assets/fac7c17f-3735-4510-ae54-499ec01d6956" />

## 概要

本スクリプトは、タイムゲートラマンによる測定データ（CSV）から
**SNR（Signal-to-Noise Ratio）に基づいて最適と考えられる時間ゲートを自動探索し、解析レポートを生成するツール**です。

蛍光バックグラウンドの影響を受けるサンプルを想定し、再現性のある時間ゲート選択を目的としています。

---

## 主な機能

### ■ SNRヒートマップによる時間ゲート最適化

* ゲート開始時間（t_start）とゲート幅を走査
* 各条件で SNR を計算
* 最大SNRとなる条件を自動選択

---

### ■ ALSベースライン補正（SNR計算時のみ）

* ALS（Asymmetric Least Squares）によるベースライン補正を選択可能
* SNR評価時のみ適用（最終スペクトルは変更しない）

プリセット：

| 設定     | 内容         |
| ------ | ---------- |
| none   | 補正なし       |
| medium | 中程度の蛍光（MF） |
| strong | 強い蛍光（SF）   |

---

### ■ ノイズ帯の自動推定

* 平坦領域を自動検出（可能な場合）
* 検出できない場合はフォールバック領域を使用
ノイズ帯はスペクトル中の平坦領域を自動検出して決定します。自動検出が失敗した場合は、あらかじめ定義されたフォールバック領域を使用します。

本フォールバックは波数方向（noise window）のみに適用されます。202602時点:時間ゲート選択にはフォールバック制約はなく、設定された探索範囲内でSNR最大条件が選択されます。
---

### ■ PDFレポート自動生成（A4・2ページ）

**1ページ目**

* 測定情報テーブル
* 処理条件（Method）
* Reference spectrum
* Time profile
* SNRヒートマップ（最適ゲート表示）

**2ページ目**

* SNV / Area 正規化スペクトル
* 選択ゲート情報

---

## 処理フロー

1. CSVファイル選択
2. Time-gated Raman データ読み込み
3. Reference spectrum 作成（時間平均）
4. （任意）ALSベースライン補正
5. ノイズ領域自動決定
6. ゲート条件を総当たり探索
7. SNRヒートマップ作成
8. 最大SNRゲートを選択
9. SNV / Area 正規化
10. PDFレポート出力

---

## ⚠️ 重要な注意事項

本ツールで算出される「最適時間ゲート」は：

> **現在の設定条件・探索範囲内における計算上の最適値**

です。

以下の設定に依存します：

* Signal / Noise 波数範囲
* ゲート幅候補
* MIN_TIME_POINTS
* ALSパラメータ（使用時）

したがって、

> 物理的に常に最適なゲートを保証するものではありません。
解析者による確認を推奨します。

## 推奨運用

1. まず ALS OFF（none）で確認
2. 蛍光が見える場合 → medium
3. 強い蛍光 → strong
4. 結果を比較して判断
